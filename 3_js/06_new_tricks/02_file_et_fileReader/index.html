New Tricks / Objets File & FileReader

Travailler avec des fichiers, c'est fun ! (et/ou casse tête)
Les objets File et FileReader permettent de récupérer des fichiers sur le système d'exploitation hôte du client, pour les intégrer à un formulaire ou un script.
Utile pour les uploader vers un serveur ou les éditer en local.
Encore un tricks à connaître !

DIY (#1hdecode )

Idéalement, utiliser un fichier de test assez lourd (vidéo de 200/300mo).
Implanter un FileReader pour créer un widget de type progress-bar.
Le widget indique la progression du chargement d'un fichier par le navigateur.
Pour cela, trouver l'écouteur d'évènements approprié.

Dissocier les problèmes

  1. HTML/Event JS pour choisir un fichier
  2. HTML/CSS pour animer la progress-bar
  3. File + FileReader + events récupérer les valeurs de l'animation (de 0 à 100%)
